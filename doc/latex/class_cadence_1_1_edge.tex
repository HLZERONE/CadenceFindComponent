\doxysubsection{Detailed Description}
Represents an edge in a graph connecting two components. 

\begin{DoxyVerb}   This class models an edge in a graph connecting two components. Each
   edge has a unique id, delay, and references to two components (A and
   B).
   Delay values are limited to [10, 100, 1000].
\end{DoxyVerb}
